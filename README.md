# React tax Dashboard

## How to

### Run application on your local

- Create a file named .env.local
- copy all the properties from .env file
- Specify the correct value of the REACT_APP_API_URL property

### Build

- install npm
- install node
- run the following command in the terminal

```shell
npm install
npm start
```
